Não, o aplicativo não possui nenhum tipo de vírus, ele é um aplicativo de código aberto, desde que o apk seja baixado dentro do site oficial ou dentro das releases do Github oficial, ele irá estar livre de qualquer tipo de vírus.

Sim, desde que a biblioteca principal esteja funcionando, caso ela acabe por quebrar e parar de funcionar, normalmente dentro de 1 a 3 dias ele irá ser consertado e você será notificado que existe uma nova versão do aplicativo, essa nova versão poderá ser uma atualização de fix (atualizações que consertam erros da aplicação) ou uma feature (atualizações que trazem novos recursos).
